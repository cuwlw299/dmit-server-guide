## 香港、洛杉矶、东京，选哪个节点

节点选择主要看你的使用场景。

香港节点延迟最低，从国内访问通常在30-50ms，适合对实时性要求高的场景，比如视频会议、游戏。但香港机房资源相对紧张，Pro套餐有时候会缺货。

洛杉矶节点适合需要美国IP的场景，CN2 GIA线路从国内到美西的延迟大概在150-180ms，跑国际流量比较稳。EB套餐是洛杉矶里性价比最高的选项。

东京节点介于两者之间，延迟比洛杉矶低，适合日本方向的需求，游戏加速用得比较多。
